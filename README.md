## Prerequisite

- [Go 1.16](https://golang.org/dl/)
- [Visual Studio Code](https://code.visualstudio.com/download) ... or your favourite Go IDE
## Quickstart

```shell
git clone https://github.com/umi-aty/DiceGame.git
cd DiceGame
```
## Running Game

```shell
go run main.go
```
